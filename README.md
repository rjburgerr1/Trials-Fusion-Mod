## 🛠️ Trials Fusion Mod Installation Guide

### ⬇️ **Download the Mod**

Download the latest release from the official GitHub page:

> https://github.com/rjburgerr1/Trials-Fusion-Mod/releases

---

### 📂 **Step 1: Locate the Trials Fusion Install Directory**

Find the main game folder for your installation:

* **Default Ubisoft Path:**
    `~Program Files (x86)\Ubisoft\Ubisoft Game Launcher\games\Trials Fusion\datapack`
* **Default Steam Path:**
    `~SteamLibrary\steamapps\common\Trials Fusion\datapack`

---

### 📋 **Step 2: Copy Mod Files**

Copy the following two files from the downloaded release into the directory you located in Step 1:

* `dbgcore.dll`
* `TFPayload.dll`

---

### ▶️ **Step 3: Start the Game**

Start **Trials Fusion** as you normally would. The mod should now be active.
